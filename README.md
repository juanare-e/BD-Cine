# BD-Cine
Sistema de base de datos relacional para la gestion de un cine, tratando de utilizar las suficientes tablas necesarias que puede necesitar un cine.
Con el fin de sustentar el proyecto final de la materia de bases de datos.
# 🎬 SistemaCine - Gestión de Complejo de Cines

Este proyecto consiste en el diseño e implementación de una **base de datos relacional robusta** para la gestión integral de un cine. El sistema está diseñado para administrar desde la programación de funciones y asignación de asientos, hasta el control de inventario en confitería y la facturación de ventas.

Desarrollado como proyecto final para la materia de Bases de Datos en la **Universidad ECCI**.

## 🚀 Características del Sistema

El diseño cuenta con una arquitectura normalizada que resuelve problemas de redundancia mediante tablas relacionales, cubriendo los siguientes módulos:

* **Gestión de Cartelera y Salas:** Clasificación de películas por género, control de duraciones y especificaciones de salas (2D, 3D, IMAX, VIP, 4DX).
* **Sistema de Boletería:** Control estricto de sillas por sala, asignación a funciones específicas y emisión de boletos con precios dinámicos.
* **Módulo de Confitería (Candy Bar):** Control de stock/inventario de productos de comida, combos y clasificación por tipo.
* **Facturación Integral:** Registro detallado de ventas asociando clientes, empleados cajeros, métodos de pago modernos (Efectivo, Tarjeta, Nequi) y subtotales calculados.

---

## 🛠️ Tecnologías Utilizadas

* **Motor de Base de Datos:** MySQL / MariaDB
* **Lenguaje:** SQL (DDL y DML)
